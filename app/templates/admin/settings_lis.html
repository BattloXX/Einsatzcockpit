Hinweis

Die API liefert aktuell keine Leitstellennummer. Einsätze aus dem LIS werden daher über Einsatzgrund, Adresse und Alarmstichwort innerhalb eines 3-Stunden-Fensters mit bereits über die API angelegten Einsätzen verbunden. Kommt ein Einsatz zuerst über das LIS, wird er selbst angelegt. Fahrzeugstatus (S4/S5) wird nur für Fahrzeuge übernommen, die unter Ressourcen eine LIS-ReferenceId hinterlegt haben. Liefert das LIS zu einem Fahrzeug Koordinaten (i.d.R. ab Status "am Einsatzort"), werden diese in dieselbe Positionshistorie wie die App-GPS-Positionen geschrieben — auf der Lagekarte gewinnt je Fahrzeug automatisch die zuletzt empfangene Position.
